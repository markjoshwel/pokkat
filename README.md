# Pokkat

immersive technology development and developing dynamic applications joint assignment 1,
virtual pet theme: tamagotchi-like pocket cat augmented reality mobile game

---

what we use, and what versions:

- Unity, 6000.0.62f1 ([Hub](unityhub://6000.0.62f1/f99f05b3e950), [DDL](https://unity.com/releases/editor/whats-new/6000.0.62f1#installs))

- SourceGit, latest ([Website](https://sourcegit-scm.github.io/)) \
  _alternative to GitHub Desktop since <https://forge.joshwel.co> is used as the primary git forge_

- sota staircase ReStepper and SideStepper \
  _(bundled in repo) repository sync script, handled by a github actions ci workflow_

- arcoreimg, from [ARCore SDK for Android v1.51.0](https://github.com/google-ar/arcore-android-sdk/releases/tag/1.51.0) \
  _(bundled in repo) to verify image tracking markers have enough keypoints_
